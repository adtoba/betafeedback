# Hero screenshots

Drop two PNG screenshots here for the landing page hero phones:

| File | Screen | Used on |
|------|--------|---------|
| `hero-project.png` | Project detail (feedback + suggested bug) | Left phone |
| `hero-bugs.png` | Bug summary (open / fixed bugs) | Right phone |

## Capturing from the iOS Simulator

1. Run the app: `cd betafeedback_mobile && flutter run`
2. Sign in and open a project with sample feedback and bugs.
3. Navigate to the **project detail** screen → `Cmd + S` → save as `hero-project.png`
4. Open **Bug summary** → `Cmd + S` → save as `hero-bugs.png`
5. Copy both files into this folder.

**Tips**

- Use full-device simulator captures (`Cmd + S`) — the site shows them as-is, no extra phone frame.
- iPhone 15 Pro (or similar) works well for aspect ratio.
